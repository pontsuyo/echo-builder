# 子供の解釈表示機能 - 設計書

## 概要
最後に家が映る時に、各子供がどんな解釈をしたかを表現する文字列を表示する機能の設計です。

## データ構造
各子供の解釈データは以下の形式で管理します。
```javascript
{
  childId: "子供のID",
  interpretation: "解釈の文字列"
}
```

## 実装詳細

### 1. データ管理
- 解釈データは `game-data.js` に追加します。
- データは配列として管理し、各子供の解釈を格納します。

### 2. 表示処理
- 家が最後に映るタイミングで、解釈データを取得します。
- 解釈データを画面上に表示します。
- 表示処理は `game-engine.js` に追加します。

### 3. UI表示
- 解釈データは画面上にテキストとして表示します。
- 表示位置やスタイルは既存のUIシステムに準拠します。

## 実装手順
1. `game-data.js` に解釈データを追加します。
2. `game-engine.js` に表示処理を追加します。
3. 表示処理を呼び出すタイミングを設定します。

## テスト
- 解釈データが正しく表示されることを確認します。
- 表示タイミングが正しいことを確認します。
- パフォーマンスに影響がないことを確認します。

## 依存関係
- `game-data.js`: 解釈データの管理
- `game-engine.js`: 表示処理の実装
