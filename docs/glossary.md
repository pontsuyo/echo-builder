# 用語集 - Echo Builders

## 1. 概要

この文書は、Echo Buildersプロジェクトで使用される用語の定義と説明を提供します。プロジェクト関係者間のコミュニケーションを明確にし、一貫性を確保するための参考資料です。

## 2. プロジェクト固有用語

### 2.1 コアコンセプト

| 用語 | 定義 | 関連用語 |
|------|------|----------|
| **Echo Builders** | 音声指示で動く、曖昧理解かわいい建築ワーカーたちの世界を提供するゲーム | Echo Workers, 曖昧理解 |
| **Echo Workers** | プレイヤーの音声指示を曖昧に理解しながら建築を進めるAIキャラクター | ワーカー, 理解確率 |
| **曖昧理解** | 完全ではなく、少しズレた理解。ゲームの核心メカニズム | 理解確率, 復唱 |
| **理解確率** | ワーカーが指示を正しく理解した確率（0-1の値） | 曖昧理解, 確信度 |
| **復唱** | ワーカーが理解した内容を音声で報告すること | 理解確率, ElevenLabs |

### 2.2 ゲームメカニズム

| 用語 | 定義 | 関連用語 |
|------|------|----------|
| **音声指示** | プレイヤーがマイクを通じて出す建築指示 | 音声認識, 転写 |
| **建築パラメータ** | 屋根形状、壁色、窓の数など、建物を定義するパラメータ | 屋根形状, 壁色 |
| **屋根形状** | 建物の屋根の形状（round, triangle, flat） | 建築パラメータ |
| **壁色** | 建物の壁の色（16進数カラーコード） | 建築パラメータ |
| **理解度スコア** | プレイヤーの指示と完成物の一致度（0-100%） | スコアリング |
| **ワーカー幸福度** | 曖昧すぎる指示によるワーカーの混乱度 | スコアリング |
| **建築美観ボーナス** | 偶然の誤解で美しい建物ができた場合のボーナス | スコアリング |

### 2.3 ワーカー属性

| 用語 | 定義 | 関連用語 |
|------|------|----------|
| **ワーカープロファイル** | ワーカーの個性と能力を定義するデータ | 理解バイアス |
| **理解バイアス** | ワーカーの理解特性（数字、色、形状、抽象語への理解力） | 数字理解力, 色理解力 |
| **数字理解力** | ワーカーが数字を正しく理解する能力（0-1） | 理解バイアス |
| **色理解力** | ワーカーが色を正しく理解する能力（0-1） | 理解バイアス |
| **形状理解力** | ワーカーが形状を正しく理解する能力（0-1） | 理解バイアス |
| **抽象語耐性** | ワーカーが抽象的な表現を理解する能力（0-1） | 理解バイアス |

### 2.4 ゲームモード

| 用語 | 定義 | 関連用語 |
|------|------|----------|
