# Building Parts SVG

建物パーツ用のSVGバリエーション集です。

- 生成日: 2026-03-01
- 総パターン数: 38
- カテゴリ: roof, wall, door, window, chimney, column

## ファイル

- `manifest.json`: カテゴリ別の一覧
- `catalog.svg`: 全パターンを俯瞰するカタログ画像
- `<category>/*.svg`: 実際のパーツSVG

## 利用例

```html
<img src="/images/building-parts/roof/roof-triangle-red.svg" alt="roof" />
<img src="/images/building-parts/window/window-lattice-cream.svg" alt="window" />
```
